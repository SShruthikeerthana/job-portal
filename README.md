Intelligent Job Portal with Job Fit Analysis
Overview

This project is a full-stack job portal that allows users to explore and apply for job opportunities. In addition to the core functionalities, it includes a feature that evaluates how well a candidate’s profile matches a job role.

The system analyzes user skills against job requirements and provides insights such as match score, missing skills, and overall assessment. This helps users understand their chances and identify areas for improvement.

Features
User registration and login
Job listing and application
Match score calculation based on skills
Identification of missing skills
Risk assessment (Low, Medium, High)
Suggestions to improve profile
Key Idea

Most job portals only display available jobs. This project adds an extra layer by helping users understand their suitability for a role. By highlighting gaps in skills and providing a match score, the system gives users a clearer idea of where they stand and what they need to improve.

Tech Stack
Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express.js

Database: MongoDB

Running the Project
Clone the repository
git clone https://github.com/SShruthikeerthana/job-portal.git
Navigate to the project folder
cd trial3
cd job portal
cd backend
Install dependencies
npm install
Start the server
npm start
Open in browser
By default, the server runs on:
http://localhost:5000
